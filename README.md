# 1collective

1collective is a back-office operations platform for blue-collar businesses — construction, HVAC, plumbing, electrical, landscaping, remodeling, and the trades broadly. The long-term product will handle the full back end of running one of these businesses: jobs, customers, scheduling, invoicing and payments, estimates, employees and timesheets, materials, and documents. It is multi-tenant — each business is its own org.

## Run locally

```bash
npm install
npm run dev
```

The app runs at http://localhost:5000.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
